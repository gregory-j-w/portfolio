Breakout
Technologies: JavaScript, Canvas, CSS, HTML

JavaScript-based game where the player controls a ball using a paddle that moves left or right along the bottom of the game board. The objective is to direct the ball to strike the bricks, which are destroyed when struck. The user wins when all the bricks are destroyed. The player looses a ball when it goes past the paddle at the bottom of the screen. The game is over when the user has used all available balls.

Stool Hopper
Technologies: Handlebars, JavaScript, CSS, Node.js, Express, Mongoose

Full stack app to expereince a truly random bar hopping experience. Users create a profile or log in, then search their surroundings area for a "random bar". The bar selection is based on the user's location and a search paramater. Once a bar has been selected, the user must visit that bar and check in - the app compares the user's location to the location in the bar's API data. If the user is within range of the bar's location, the check in is successfule and the user can search for their next stop.
